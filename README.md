# Jawad Superstore Sales Analysis Dashboard 📊

A modern and interactive executive dashboard built with HTML, CSS, JavaScript, and Chart.js for analyzing US Superstore sales data (2018–2022). The dashboard provides deep business insights through dynamic visualizations, KPI tracking, heatmaps, trend analysis, and filtering features using Excel/CSV-based datasets.

## ✨ Features

* Interactive KPI cards
* Monthly sales & profit trends
* Category and sub-category analysis
* Regional performance insights
* Customer segment analysis
* Discount impact visualization
* Sales vs profit bubble charts
* Shipping mode analytics
* Year-over-year growth tracking
* Monthly order heatmap
* Responsive modern UI design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Chart.js
* Excel / CSV Data Integration

## 📂 Dataset

The dashboard uses Superstore sales transaction data containing:

* Orders
* Revenue
* Profit
* Discounts
* Customer segments
* Shipping modes
* Regional sales performance

## 🚀 How to Run

1. Download or clone the repository
2. Open the `.html` file in your browser
3. Explore the interactive dashboard

## 📈 Purpose

This project was created to demonstrate business intelligence, sales analytics, and data visualization skills using front-end web technologies.
